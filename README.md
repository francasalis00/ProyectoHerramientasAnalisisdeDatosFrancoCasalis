# 📊 Análisis de Ventas Online Retail

**Autor:** Franco  
**Curso:** Herramientas para el análisis de datos  

---

## 🎯 Objetivo

Analizar un dataset de ventas online para entender el comportamiento de los ingresos, productos y países.

---

## 📦 Dataset

Dataset de ventas online con información de productos, cantidades, precios y países.

---

## 🛠️ Proceso

- Limpieza de datos  
- Eliminación de nulos y duplicados  
- Creación de revenue  
- Análisis con Python  
- Dashboard en Power BI
